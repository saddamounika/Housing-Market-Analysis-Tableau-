# Housing-Market-Analysis-Tableau-
📌 Project Overview
This project presents a comprehensive analysis of the housing market trends for ABC Company using a dataset of over 21,000 house sales. The goal was to build an interactive dashboard that provides insights into pricing, renovation impacts, and house age trends.

🛠️ Technical Architecture & Flow
The project follows a structured data science workflow:

Data Preparation: Utilizing Python for cleaning raw house data and performing feature engineering.

Data Visualization: Developing specific worksheets to analyze price distributions and property conditions.

Dashboarding: Creating an interactive environment in Tableau to filter and explore data dynamically.

💡 Key Business Insights
Market Volume: Analyzed 21,613 total houses with an average sale price of $540,088.

Renovation Trends: Only 4.23% of the houses in the dataset are renovated, while 95.77% remain in their original condition.

Price Distribution: Most houses are concentrated in the lower price brackets, as shown in the Price Bin histogram.

Age vs. Value: There is a clear fluctuation in average prices based on the house age, with older homes showing distinct market behavior.

📊 Final Dashboard
The final interactive dashboard combines all metrics into a single view for stakeholders:

Top Left: Key Performance Indicators (KPIs).

Center: Price Distribution Histogram & Renovation Pie Chart.

Right: Average Price vs. House Age Line Chart.
